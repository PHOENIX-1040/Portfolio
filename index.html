§ 02 — Selected Work

Four projects.

01. 2025 IEEE Publication UAV · ML

Drone-Based Precision Agriculture using Machine Learning

A solar-powered DJI Mavic 2 Pro framework that captures 4K aerial imagery of farmland and routes it through cloud-based deep learning pipelines to automate four critical tasks: water-scarcity detection, weed mapping, crop disease identification with treatment recommendations, and growth-stage yield prediction. The system delivers real-time insights to farmers through a mobile interface, replacing the cost and rigidity of conventional IoT-only setups.

02. 2025 Cybersecurity Real-time ML

AI-Driven Network Security Dashboard

A hybrid intrusion detection framework that unifies UDP Flood (DDoS) and ARP Spoofing detection into a single real-time monitoring dashboard. Two Random Forest classifiers run in parallel on independently engineered feature sets, with their outputs fused at the dashboard layer to visualise risk probabilities, alert urgency, and live traffic analytics — replacing the brittle signature-matching of traditional IDS.

03. 2024 NLP · RAG OOSE

Data RAG — Retrieval-Augmented Document Intelligence

A modular retrieval-augmented generation framework that ingests heterogeneous corpora (PDFs, text, structured sources), chunks and embeds them via Sentence-BERT / MiniLM, and serves semantic queries through a FAISS vector index with cosine similarity. The architecture cleanly separates ingestion, embedding, vector storage, query handling, and UI — designed as an OOSE artifact with full test coverage for ambiguity, edge-case data, and real-time responsiveness.

04. 2026 Final Year Project Vision Transformers

Intelligent Violence Detection with DeiT & Temporal Consistency

A real-time campus surveillance system built on a Data-Efficient Image Transformer (DeiT) backbone with a temporal-consistency layer for smoothing frame-level predictions into reliable event detections. The goal: overcome the well-documented attention decay of human CCTV operators (which drops sharply after 15–20 minutes) and the chronic false-positive rate of motion-based heuristics — flagging genuine threats with low latency while keeping benign activity quiet.
